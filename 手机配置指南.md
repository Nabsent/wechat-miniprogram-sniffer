# 手机代理配置指南

## 📱 Android 手机配置

### 方法一：WLAN 代理设置（推荐）

1. **打开设置** → **WLAN/WiFi**
2. **长按**已连接的 WiFi 名称
3. 选择 **修改网络** 或 **网络详情**
4. 展开 **高级选项**
5. **代理** 选择 **手动**
6. 填写配置：

```
代理服务器主机名:  192.168.1.100  (填写电脑的 IP)
代理服务器端口:    8888
绕过代理:          (留空)
```

7. 点击 **保存**

### 不同 Android 版本的差异

#### 小米/红米 (MIUI)
- 设置 → WLAN → 点击已连接的 WiFi
- 代理 → 手动
- 主机名 + 端口

#### 华为/荣耀 (HarmonyOS)
- 设置 → WLAN → 点击已连接的 WiFi
- 向下滑动找到"代理"
- 手动 → 填写服务器地址 + 端口

#### OPPO/一加 (ColorOS)
- 设置 → WLAN → 点击已连接的 WiFi 右侧的齿轮
- 代理 → 手动代理
- 服务器主机名 + 端口

#### vivo (OriginOS)
- 设置 → WLAN → 点击已连接的 WiFi
- 代理 → 手动
- 主机名 + 端口

## 📱 iPhone/iPad 配置

1. **设置** → **无线局域网**
2. 点击已连接 WiFi 右侧的 **(i)** 图标
3. 向下滑动到底部，找到 **HTTP 代理**
4. 选择 **手动**
5. 填写配置：

```
服务器:  192.168.1.100  (填写电脑的 IP)
端口:    8888
鉴定:    关闭
```

6. 点击右上角 **存储**

## ❓ 常见字段说明

| 字段名称 | 填写内容 | 说明 |
|---------|---------|------|
| **主机名/服务器/代理服务器地址** | `192.168.1.100` | 电脑的 IP 地址 |
| **端口/代理服务器端口** | `8888` | 固定填 8888 |
| **用户名/密码/鉴定** | 留空 | 不需要填写 |
| **绕过代理/排除列表** | 留空 | 不需要填写 |
| **网关/默认网关** | 保持默认 | 通常是 `192.168.1.1` |
| **DNS** | 保持默认 | 不需要修改 |

## 🔍 如何获取电脑 IP

### 方法一：使用工具（推荐）
```cmd
python get_ip.py
```

### 方法二：命令行查看

**Windows:**
```cmd
ipconfig
```
找到"无线局域网适配器 WLAN"下的"IPv4 地址"，类似 `192.168.1.100`

**Mac/Linux:**
```bash
ifconfig | grep "inet "
```
或
```bash
ip addr show
```

### 方法三：启动代理时自动显示
运行 `python proxy_sniffer.py` 时会自动显示：
```
[*] 请配置手机代理为: 192.168.1.100:8888
```

## ⚠️ 注意事项

### 1. 如果填写框有"网关"选项
- **保持原来的值**不要改（通常是路由器地址，如 `192.168.1.1`）
- 或者**留空**
- 网关和代理是两回事，不会影响抓包

### 2. 确认在同一网络
- 手机和电脑必须连接**同一个 WiFi**
- 不能是手机用 WiFi，电脑用网线（除非在同一局域网）

### 3. 防火墙问题
如果配置正确但无法连接，检查电脑防火墙：

**Windows 防火墙:**
```cmd
# 临时关闭防火墙测试（不推荐长期关闭）
或者添加入站规则允许 8888 端口
```

**允许 Python 通过防火墙:**
- 控制面板 → Windows Defender 防火墙 → 允许应用通过防火墙
- 找到 Python，勾选"专用"和"公用"

### 4. IP 地址选择
如果电脑有多个 IP（多网卡），优先选择：
- `192.168.x.x` 格式的（家庭/办公室网络）
- 不要选 `127.0.0.1`（本机回环）
- 不要选 `169.254.x.x`（自动分配，说明网络有问题）

## 🧪 测试连接

配置好代理后：

1. **打开手机浏览器**
2. 访问 `http://www.baidu.com`
3. 如果能打开网页，说明代理配置成功
4. 电脑终端应该显示捕获到的请求

如果无法上网：
- 检查 IP 地址是否正确
- 检查代理服务器是否在运行
- 检查防火墙设置

## 📋 快速配置检查清单

- [ ] 电脑和手机连接到同一个 WiFi
- [ ] 已启动代理服务器 (`python proxy_sniffer.py`)
- [ ] 记录了显示的 IP 地址（如 `192.168.1.100`）
- [ ] 手机代理设置为"手动"
- [ ] 主机名/服务器填写了正确的 IP
- [ ] 端口填写了 `8888`
- [ ] 用户名/密码留空
- [ ] 网关保持默认或留空
- [ ] 用浏览器测试代理是否生效

## 🎯 完整流程示例

```
1. 电脑运行: python proxy_sniffer.py
   显示: [*] 请配置手机代理为: 192.168.1.100:8888

2. 手机配置:
   设置 → WLAN → 点击已连接的 WiFi
   代理 → 手动
   主机名: 192.168.1.100
   端口: 8888
   保存

3. 手机浏览器访问 baidu.com 测试
   电脑终端应显示捕获的请求

4. 打开微信小程序，触发文件下载
   电脑终端显示: [+] 捕获文件: xxx.pdf

5. 电脑按 Ctrl+C 停止代理

6. 电脑运行: python file_downloader.py
   选择要下载的文件
```

完成后记得**关闭手机代理**恢复正常上网！
